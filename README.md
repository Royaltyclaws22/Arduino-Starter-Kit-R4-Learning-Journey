# 🤖 Arduino Starter Kit R4 Learning Journey

This repository documents my journey learning electronics and programming with the Arduino Starter Kit R4.

The projects in this repository are based on the official Arduino Starter Kit R4 projects and documentation. 
I have built the circuits myself, captured my own photos, written my own explanations, and adapted or rewritten the code while experimenting with different implementations.

The purpose of this repository is educational: to document my learning process, my understanding of each circuit, and my ideas for future improvements of each project.
